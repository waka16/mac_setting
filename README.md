# mac_setting

## Preparation

```
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Install

```
$ cd
$ git clone https://github.com/waka16/mac_setting.git
$ cd ~/mac_setting
$ ./mac_default.sh
$ ./homebrew.sh
```
